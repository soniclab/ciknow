Question Types

Rating questions ask users to rate statements on a given scale.

Choice questions allow users to select one or multiple answers from a list of possible answers.

Text questions allow for the creation of short-answer or essay-style questions by filling in text fields.

A contact info question asks users to fill in fields for their contact information.

Relational choice questions are used to gather data on how various user groups, or classes, interact with each other.  In general, a relational choice question asks the participant how they relate to other people based on specified criteria. .

Perceived rating questions allow users to rate statements on a given scale for other users (also referred to as cognitive rating questions). 

Perceived choice questions allow users to choose statements for other users (also referred to as cognitive choice questions). 

Perceived relational rating questions are used to gather data on how users connect to other users and gather data based on how a user thinks other users relate to each other . 

A display page shows an information only page for a CI-KNOW survey.

A contact chooser allows users to select their contacts.  Essentially, it allows users to select a subset of user or add new users into a survey.
